# SignBridge 🤝

**AI-powered American Sign Language (ASL) recognition system** that translates hand gestures into readable English text in real-time — right in your browser.

🔗 Live demo: [sign-sense-flow.lovable.app](https://sign-sense-flow.lovable.app)

## What it does

SignBridge helps bridge communication between deaf/mute individuals and others by recognizing ASL hand signs through a camera and converting them into text — no sign-up required, runs entirely in the browser.

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Hand gesture recognition
- Built with [Lovable](https://lovable.dev)

## Features

- Real-time hand gesture detection
- Converts ASL signs to English text
- No sign-up or account needed
- Runs 100% locally in the browser
- Simple, accessible UI

## Getting Started

```bash
npm install
npm run dev
```
